Internationalization (i18n) Documentation

This documentation provides an overview of how internationalization is implemented in the Voilà Vélo website. The website supports multiple languages (currently French and English) through a custom React Context-based solution.

Quick Start

To use translations in your component:

// Import the hook
import { useI18n } from "@/lib/i18n";

// Inside your component
const { t, language, setLanguage } = useI18n();

// Use translations
<h1>{t('page.title')}</h1>
<p>{t('page.description')}</p>

// Switch language
<button onClick={() => setLanguage('en')}>English</button>
<button onClick={() => setLanguage('fr')}>French</button>

Examples

1. Basic Translation

Translation Key: navigation.home

French Value: "Accueil"

English Value: "Home"

Usage: <Link href="/">{t('navigation.home')}</Link>

2. Complex Content

Translation Key: whyChoose.reasons.sustainable.description

French Value: "Nos vélos fonctionnent sans électricité et nous utilisons uniquement des matériaux durables et biodégradables."

English Value: "Our bikes operate without electricity and we only use sustainable and biodegradable materials."

Usage: <p>{t('whyChoose.reasons.sustainable.description')}</p>
